배열필수함수(자동화)

database서버에서 응답받은 데이터를 2차적으로 자바스크립트 표현식에 맞게 수정하기 위해서는 배열을 자유롭게 다룰 줄 알아야함

index

  • push()

push()

배열 끝에 원소를 추가

pop()

배열의 마지막 원소를 제거

concat()

두개 혹은 더많은 배열을 합칠 때 사용

원래 사용한 배열은 바꾸지 않고 새로운 배열을 만들어서 리턴하는 함수

reverse()

배열의 순서를 뒤집어주는 함수

join()

배열을 문자열로 리턴하는 함수

join()함수의 인자로는 구분자(문자)를 넣어준다. (구분자가 없다면 기본값이 콤마(,)로 설정되어 있다.

join()함수는 원본 데이터를 수정하지않고 계산된 새로운 문자를 리턴하는 함수

split()

문자열을 배열로 리턴하는 함수

문자데이터를 인자로 사용된 특수한 구분문자로 쪼개서 배열로 리턴하는 함수

원본데이터는 수정하지 않고 계산된 새로운 배열을 리턴하는 함수

splice

선택한 배열 인덱스를 통해 원하는 갯수만큼 선택한배열에서 원소를 제거하여 배열형태로 리턴하는 함수

배열.splice(시작인덱스,제거할원소의 갯수,추가할 원소...)

slice

선택한 배열의 인덱스(시작,끝)을 통해 특정원하는 부분을 새로운배열리턴하는 함수

시작인덱스만 입력한경우는 시작인덱스부터 배열끝까지 쭉 새로운배열리턴

map

배열에 사용할 수 있는 함수

배열에 속한 각각의 원소에 대해서 입력한 콜백함수를 실행하고 그 결과로, 새로운 배열을 리턴하는 함수

map이라는 함수는 매개변수로 콜백함수를 받는다

map에 사용되는 콜백함수의 매개변수는 첫번째 값은 map함수를 사용한 배열의 원소의 값이 저장된다

map에 사용되는 콜백함수 선택배열.map(function(선택배열의값,선택배열의순번,원본선택배열){})

find

find함수는 선택한 배열에서 특정 조건을 만족하는 원소를 찾아서 리턴하는 함수

find함수는 리턴의 조건에서 조건에 만족되는 원소가 한개만 리턴된다(원소의 형태로)

find함수는 원소의 작은순번부터 차례대로 탐색하기 때문에 조건에 맞는 원소를 찾으면 종료된다

filter

find함수가 특정조건에 맞는 원소 한개를 선택한 것이라면 filter함수는 사용방식은 find함수와 똑같지만 특정조건에 맞는 원소 여러개를 새로운 배열을 만들어서 리턴해주는 함수

forEach

선택한 배열의 각 원소에대해 입력한 콜백함수를 실행시켜주는 함수

따로 리턴하는 값이 없고 실행만 시킨다
